Where can I get the source code for the PhET simulations?

To browse PhET source code online

  1. Visit the URL https://phet.unfuddle.com/a#/repositories/23262/browse?path=/trunk
  2. Enter username=guest and password=guest

Navigating our source code

Everything is under the folder "trunk", and README.txt describes its folder structure. Simulations can be found under these folders:

  • Java simulations: simulations-java/simulations
  • Flash simulations: simulations-flash/simulations
  • Flex simulations: simulations-flex/simulations

PhET simulations are organized into projects, each of which can contain one or more related simulations. For example, the "nuclear-physics" project contains 4 simulations, including "Alpha Decay" and "Radioactive Dating Game".

To checkout PhET's source code repository using Subversion on the command-line

Use the following command: 

svn checkout https://phet.unfuddle.com/svn/phet_svn/trunk trunk --username guest --password guest

This will put all files in a folder named "trunk" -- it will take a while, so please be patient.

To build and run PhET simulations using the PhET Build GUI

Check out our source code with the instructions above, and then run one of these two scripts depending on your operating system:

Windows:
build-tools/build.bat
Mac, Linux:
build-tools/build.sh

Using the PhET Build GUI

  1. For building Flash/Flex simulations, please download build-local.properties to trunk/build-tools/build-local.properties, and replace relevant paths with your own.
  2. Browse through and select the project (listed on the left) that contains the simulation you are looking for.
  3. Select the simulation and/or locale
  4. Press "Test". Be patient, since it can take several minutes to compile and launch the simulation.
