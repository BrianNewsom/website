Metadata Links

It is recommended that if people need up-to-date metadata, they should harvest from our OAI-PMH provider. Four metadata formats are available (Dublin Core, NSDL Dublin Core (NSDL_DC), IEEE LOM, and IMS LODE ILOX).

  • The OAI-PMH provider is available at http://phet.colorado.edu/oai/provider
  • jOAI administration is available at http://phet.colorado.edu/oai/admin/ (this is the easiest way to view metadata formats. see /usr/local/tomcat6/webapps/oai/WEB-INF/users.xml on figaro for user credentials)
  • Metadata made available to North Carolina is available at http://phet.colorado.edu/services/nc-info.csv
  • Metadata made available to Learning.com is available at http://phet.colorado.edu/services/learning-com-export.csv
  • An up-to-date simulation list is available at http://phet.colorado.edu/services/project-sorted-simulations.csv
  • A (very large) dump of all simulations in the PhET master-format is available at http://phet.colorado.edu/metadata-dump.xml (this has all data included in other metadata)
