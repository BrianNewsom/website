Student Use

Why should I use this simulation with my students?
Our studies show that using the simulation helps make complex circuit concepts and lab skills easier for students. To find out more, check out these articles: Assessing the effectiveness of computer simulation in an introductory undergraduate environment and Assessing the effectiveness of a computer simulation in conjunction with Tutorials In Introductory Physics in undergraduate physics recitations
Are there any common student difficulties that come up when students interact with the simulation?
Students sometimes have difficulty recognizing if the switch is completely closed or not.

General

What's the underlying model being used?
The short answer is, the model is computed using Modified Nodal Analysis. Capacitors and inductors are linearized at each time step and dynamic timestep subdivisions are used to balance accuracy and speed.
If you're interested in more details, email questions to phethelp@colorado.edu.
Why are there multiple versions of Circuit Construction Kit?
The DC version of Circuit Construction Kit is the simplest. This is a good simulation to get your students started. The AC version includes AC voltage sources, capacitors and inductors. The "Virtual Lab" versions do not have the ammeter that can be moved onto components to determine the current. This feature was added at the request of teachers who use ammeters that must be incorporated into the circuit.

Play Area

How do I make a circuit?
Drag components from the toolbox and put them together!
How do I break a junction or remove a component?
To break a junction, right click over that junction. A dialogue box will appear, select 'Split Junction'.
To remove a component, right click on that component. A dialogue box will appear. Select 'Remove'.
Can I stop the sim?
Sure! Use the Pause button at the bottom of the simulation to pause the sim. Then select Play to when you're ready to keep going. You can also use the Step button to see what's happening in small time increments.
What's the best screen resolution to use during lecture?
If you set your screen resolution to 1024x768, the simulation will fill the full screen.
Can I change the background/toolbox color for easier viewing/printing?
You sure can! Just select 'Options' in the upper right hand corner of the simulation, select 'Background Color', pick the color you prefer (white works well for printing) and select 'OK'. If you want to change it back to the original settings, follow the same instructions, but select 'Reset' instead of 'OK'.
Why does my circuit look like it's on fire?
Because it is! Fire denotes a short circuit, or a very high current.
Why do the electrons not appear to be moving, even though I know the circuit is working (the lightbulb is lit, etc)?
You probably have created a circuit with a very high current. When this happens, the blue electron dots cannot be shown to move fast enough. When this happens, the electrons appear to 'freeze' and the text "Animation speed limit reached! Simulation speed reduced to 75% normal!" is shown in red on the lower right hand side of the screen.

Circuit Toolbar

Can I save a configuration for later use?
Yes! Use the Save button to save a configuration for lecture or homework.
How do open a saved configuration?
To open a saved configuration, download the file. To open a file, the Circuit Construction Kit simulation must be open. Press the 'LOAD' button, and load the file with the .cck extension. If the saved configuration file does not have the .cck extension, then in the FILE TYPES box, select 'ALL FILES'.
Can I start over?
Use the 'Reset' button to restart the sim to its original settings.

Visual Toolbar

What's the difference between Lifelike and Schematic?
The lifelike option shows a cartoon-like version of all of the components. This is a great place to stay with younger students, or for introductory classes. The schematic option shows the components as you would see them drawn in schematic representations of circuits. This is great for more advanced students.
Is there a way to see the number value of the resistance or voltage of a component?
Sure! Just select Show Values.

Tools Toolbar

Can I determine what the actual value of the wire resistivity is?
Sure! Measure the current and voltage values, and use Ohm's Law R = V/I to determine the wire resistance. Note, when you change wire resistivity, the amount of resistance will vary with the length of the wire.

How do I change the resistance or voltage of a battery?
To change the internal resistance or voltage of the battery, right click on the battery and select the appropriate option from the drop down menu.

Grab Bag

What's the difference between the hand and the dog in the Grab Bag?
The dog's hair is a good insulator, while the hand is a good conductor (not much hair there!).
